About App

• The main purpose of this Application is to receive and display the output data from navigation systems using Bluetooth connection. In real time, the pilot can receive information about the orientation and coordinates of the aircraft.

• Support vectors tails from MapBox API

• The app has a mobile and tablet version

• also the application can record output files from the navigation system

Connection

For the application to work correctly, you must use special navigation systems with a bluetooth module. Application architecture easily adapts to new systems.

Dashboard

The application implements a model of a real aircraft dashboard.

Satellites

The app provides information about the current satellites. And shows DOP information
